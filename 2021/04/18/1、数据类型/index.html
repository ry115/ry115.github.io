数据类型

发表于 2021-04-18

##强类型语言

​ Java是一种强类型语言：要求变量的使用严格符合规定，所有变量必须先定义后才能使用

Java的数据类型分为两大类型

基本类型（primitive type）

数值类型

整数类型

  • byte ：占1个字节范围，-128~127

  • short ：占2个字节范围，-32768~32767

  • int ：占4个字节范围，-2,147,483,648~2,147,483,647

  • long ：占8个字节范围，（定义该类型时，注意在数值后加上“L”）

浮点类型

  • float ：占4个字节（定义该类型时，注意在数值后加上“f”）

  • double ：占8个字节

字符类型

  • char：占2个字节

    在Java中，char类型本质是一个整数，在输出时，是Unicode码对应的字符。所以我们可以直接给char类型赋值一个整数，然后输出时会按照对应Unicode表输出对应字符。

  • 常用的编码类型

    1. ASCLL码：使用一个字节来表示符号，虽然可以表示256个，但只使用了128个
    2. Unicode编码：使用两个字节来表示，可以表示2^16^=65536个字符，缺点是会造成存储空间的浪费
    3. UTF-8编码:英文字母用一个字节来表示，汉字用3个字节来表示

Boolean类型

占1位，值只有true和false两个

引用类型（reference type）

  • 类
  • 接口
  • 数组

什么是字节

  • 位（bit）：是计算机内部 数据存储 的最小单位，例如11001100是一个8位二进制数。
  • 字节（byte）：是计算机中 数据处理 的基本单位，习惯上用大写B来表示。
  • 字符：是指计算机中使用的数字、字母、字、符号。
  • 1B（字节）= 8 bit（位）
  • 1KB = 1024B
  • 1M = 1024KB
  • 1G = 1024MB
